# F15 Femkamp Poängräknare

Score calculator for F15 girls pentathlon in Swedish athletics.

## Features

- Toggle between indoor and outdoor pentathlon events
- Real-time point calculation using World Athletics formulas
- Mobile-optimized interface
- Save and load results using browser localStorage

## Free Hosting with GitHub Pages

1. Create a GitHub repository
2. Push this code to the repository
3. Go to repository Settings → Pages
4. Set Source to "Deploy from a branch" and select `main`
5. Your site will be live at `https://yourusername.github.io/repositoryname`

## Usage

1. Select Indoor (Inomhus) or Outdoor (Utomhus) mode
2. Enter results for each event
3. Points are calculated automatically
4. Tap "Spara resultat" to save locally
5. Previously saved results appear at the bottom
